# Python-MicrosoftOffice-ShapeControl
PythonでMicrosoftOffice（Excel、Word、PowerPoint）での図形操作

## 実行環境
- Windows10 Pro：21H1
- Microsoft Office：2019
- Python：3.9.6

## インストールするライブラリ
- pywin32
```powershell
> pip install pywin32
```
## 実行方法
コマンドプロンプトから下記を実行
```powershell
> python shape_control.py  "対象フォルダパス"
```
